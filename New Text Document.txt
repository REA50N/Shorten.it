# Shorten.it

A modern, fast, and beautiful URL shortener built with **Next.js 15**, **TypeScript**, **Prisma**, and **Tailwind CSS**.

![Preview](https://via.placeholder.com/800x400?text=Shorten.it+Preview)

## ✨ Features

- **Custom Short URLs** – Create memorable slugs (e.g., `shorten.it/my-link`)
- **Click Analytics** – Track total clicks and performance
- **QR Code Generation** – Downloadable QR codes for every link
- **Beautiful Dashboard** – Modern glassmorphism UI with collapsible sidebar
- **User Authentication** – Secure login with Google (NextAuth)
- **Responsive Design** – Works perfectly on desktop and mobile
- **Real-time Search** – Filter your links instantly
- **Dark Mode** – Sleek dark theme with premium feel

## 🚀 Tech Stack

- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Prisma + PostgreSQL
- **Auth**: NextAuth.js (Google Provider)
- **UI Components**: shadcn/ui
- **Icons**: Material UI Icons
- **Styling**: Tailwind + Custom Glassmorphism

## 🛠️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/shorten-it.git
cd shorten-it
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file in the root:

```env
DATABASE_URL="your-postgresql-url"
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"
GOOGLE_CLIENT_ID="your-google-client-id"
GOOGLE_CLIENT_SECRET="your-google-client-secret"
```

### 4. Database Setup

```bash
npx prisma generate
npx prisma db push
# or
npx prisma migrate dev
```

### 5. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
